****Executar un fitxer java pels exercicis 1-3 desde la línia de comandes****

Des d'una terminal Git Bash, amb codificació UTF-8 (es pot crear des del propi Eclipse):

1. Situar la terminal en el paquet n1exercici<número d'exercici> (per exemple, n1exercici1)
2. Executar la següent línia de comandes: java N1_E<número d'exercici>_main.java <directori on es vol fer la cerca, utilitzant barres / i no \>
   Per exemple, java N1_E1_main.java D:/Documentos/Estudis

****Executar un fitxer java per l'exercici 4 desde la línia de comandes****

Igual que en el cas anterior, afegint a la direcció del fitxer el seu nom amb l'extensió. Per exemple, java N1_E4_main.java D:/Documentos/fitxer.txt

****L'exercici 5 no s'executa desde la línia de comandes, sinó desde l'IDE****
