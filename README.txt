****Executar un fitxer java desde la línia de comandes****

Des d'una terminal Git Bash, amb codificació UTF-8 (es pot crear des del propi Eclipse):

1. Situar la terminal en el paquet n1exercici1
2. Executar la següent línia de comandes: java N1_E1_main.java <directori on es vol fer la cerca, utilitzant barres / i no \>

Un exemple de directori seria D:/Documentos/Estudis, que col·locat a la línia de comandes seria així:
java N1_E1_main.java D:/Documentos/Estudis
